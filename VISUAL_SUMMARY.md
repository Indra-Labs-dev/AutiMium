## 🎓 Example: Nmap Scan

**User Flow:**
```
1. Open "Kali Tools"
2. Click "Scanning"
3. Select "nmap" card
4. Enter target: 192.168.1.1
5. Set ports: 1-1000
6. Click "Start Scan"
```

**Backend Execution:**
```bash
nmap -p 1-1000 192.168.1.1
```

**Output Displayed:**
```
[+] Scan initiated for 192.168.1.1
[+] Using tool: nmap
[+] Ports: 1-1000

[*] Scanning...
PORT    STATE SERVICE
22/tcp  open  ssh
80/tcp  open  http
443/tcp open  https

Nmap done: 1 IP address (1 host up)
```
